# SHOPPINGTRENDS_EDA
Exploratory Data Analysis on Customer Shopping Trends dataset using Python, Pandas, Matplotlib and SeaboCustomer Shopping Trends - Exploratory Data Analysis (EDA)
About the Dataset

This project analyzes the Customer Shopping Trends dataset to understand customer purchasing behavior and identify patterns in shopping activities. The dataset contains customer demographics, purchase information, product categories, seasonal trends, review ratings, and other shopping-related attributes. It consists of multiple records representing individual customer purchases and shopping preferences.

Objective

The main objective of this project is to explore customer shopping behavior and uncover meaningful insights from the data. The analysis aims to answer questions such as:

Which product categories generate the highest purchase amounts?
How do shopping patterns vary across different seasons?
What is the distribution of customers based on gender and age?
Are there any relationships between product categories and customer ratings?
Are there any outliers in purchase amounts?

Python
Pandas

Analysis Performed

The following analyses were carried out:

1. Data Exploration and Quality Check
Dataset overview using shape, info, and describe.
Checked for missing values and duplicate records.
2. Univariate Analysis
Gender distribution.
Age distribution.
Season distribution.
Purchase amount distribution.
3. Bivariate Analysis
Product Category vs Purchase Amount.
Product Category vs Review Rating.
Gender vs Purchase Amount.
4. Correlation Analysis
Correlation heatmap for numerical features.
5. Outlier Detection
Boxplots used to identify unusual purchase amounts.
6. Pivot Table Analysis
Summarized purchasing trends across categories and seasons.
Key Findings
Certain product categories contributed significantly higher purchase amounts than others.
Seasonal trends influenced customer purchasing behavior.
Customer ratings varied across product categories.
The dataset contained minimal data quality issues.
Purchase amount distributions showed the presence of a few outliers.
Demographic factors such as gender provided useful insights into shopping preferences.
Conclusion

The analysis provides valuable insights into customer shopping behavior, product performance, and seasonal purchasing patterns. These findings can help businesses make data-driven decisions related to marketing strategies, inventory management, and customer engagement.
